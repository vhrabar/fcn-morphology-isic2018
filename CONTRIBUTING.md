# Contributing to the Project

This project forms part of an academic group project for [Digital Image Processing and Analysis](https://www.fer.unizg.hr/en/course/dipaa) course at the University of Zagreb, Faculty of Electrical Engineering and Computing (FER).
Only the officially assigned members of the project are allowed to contribute to this repository; all external contributions will be ignored.
---

## 1. Code of Conduct
All contributors are expected to adhere to our [Code of Conduct](CODE_OF_CONDUCT.md).

## 3. Development Setup
In order to maintain the quality and consistency of the codebase, all team members must follow the established development setup oulined below:

- Python 3.12+
- Version Control: Git with GitHub (main branch: `main`)

and other tools listed in the [requirements.txt](requirements.txt) file.
